Profil Aplikasi

Aplikasi media pembelajaran teknologi informasi pada smartphone android adalah aplikasi media pembelajaran mata pelajaran teknologi informasi komunikasi (TIK) yang dibuat kusus berbasis android.

Tujuan Aplikasi

Adapun tujuan dari pembuatan Aplikasi media pembelajaran mata pelajaran teknologi informasi komunikasi (TIK) adalah :
  • Mengenalkan perangkat-perangkat teknologi informasi kepada pelajar tingkat SMP.
  • Meminimalkan kekurang pahaman siswa SMP tehadap pengetahuan perangkat teknologi informasi.
  • Menghasilkan sebuah aplikasi media pembelajaran teknologi informasi untuk pelejar SMP.

Kegunaan Aplikasi

Aplikasi media pembelajaran teknologi informasi pada smartphone android Dapat menjadi media pemblajaran teknologi informasi untuk memotivasi belajar secara mandiri serta menumbuh kembangkan minat bakat siswa SMPIT Bani Ummah terhadap bidang teknologi informasi.
